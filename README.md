# Simple Interest Calculator

A simple Python program that calculates **simple interest** based on the principal, rate of interest, and time period in years.  
This project is designed for beginners to practice basic arithmetic operations and understand simple financial calculations.

## Features
- Calculates simple interest using the formula: `SI = P * R * T`
- Accepts user input for principal, rate, and time
- Prints the calculated simple interest

## Inputs
- `P` – Principal amount
- `R` – Annual rate of interest (in decimal or percentage)
- `T` – Time period in years

## Output
- `SI` – Simple interest calculated using the formula `SI = P * R * T`
